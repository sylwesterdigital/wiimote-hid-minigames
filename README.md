## wiimote-hid-minigames for Samsung SRUK

```
https://xr.workwork.fun/samsung/wall/minigame4/
```

## get repository
```
$ git clone git@github.com:sylwesterdigital/wiimote-hid-minigames.git
```

## run demo
```
$ cd wiimote-hid-minigames
$ npm i
$ npm run start
```

## build demo
```
$ build
```
npm run build
